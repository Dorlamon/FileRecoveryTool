
# Office Recovery Toolkit v5.8 (LightBar Menu)

使用方式:
解壓縮後，執行: OfficeRecoveryToolkit.cmd 

必要安裝:
Microsoft Office Word、Excel 及 PowerPoint 2007 檔案格式相容性套件
https://w3.nhps.tp.edu.tw/cptech/upload/FileFormatConverters.exe

  
這版包含：  
  
強化報表功能  
美化報表樣式  
選單改成光棒版本    
掃描 docx/xlsx/pptx/doc/xls  
zh-TW 語系顯示繁體中文介面，其它語系顯示英文介面 (可手動切換)  
使用 SHA256 雜湊值去重複檔案  
檔案內容指紋比對  
Excel 全部工作表解析  
Excel 檔案智慧命名  
主檔 / 重複檔案 / 唯一檔案 / 損毀檔案角色判定  
模擬檔案改名 / 實際檔案改名  
HTML 報表  
開啟最新 HTML 報表 (使用目前系統預設的瀏覽器開啟)  
安全整理模式：Copy/Move (檔案處理模式)  
Primary Only 開關  
依副檔名分類整理檔案  
Quality Score 品質分數  
HTML 顯示主檔 ⭐ Primary
